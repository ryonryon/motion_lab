# Project Overview

This is a React Native Expo app called MotionClip.

## Purpose
A mobile app for athletes to:
- record videos
- trim clips from long videos
- tag actions (spike, serve, receive)
- review performance

## Tech Stack
- React Native (Expo)
- TypeScript
- expo-av (video playback)
- ffmpeg-kit-react-native (video trimming)
- local storage (expo-sqlite or MMKV)

## Principles
- Fast and smooth UX
- Simple UI (like TikTok)
- Offline-first

## File directories
- mobile
  directory for mobile development
- packages
  directory for business logic
